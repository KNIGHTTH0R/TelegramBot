# TelegramBot


| Имя команды            | Описание функционала                                |
|------------------------|:----------------------------------------------------|
| ``/movies``            | фильмы в прокате и премьеры этой недели             |
| ``/nearest``           | ближайшие кинотеатры                                |
| ``/return``            | возврат билета                                      |
| ``/help``              | помощь по командам                                  |


| Синонимы               | Команда                  |
|------------------------|:-------------------------|
| ``рядом, ближайшие``   | ``/nearest``             |
| ``фильмы``             | ``/movies``              |
| ``возврат``            | ``/return``              |
| ``помощь``             | ``/help``                |



### доступные выражения: 

#### ... (@фильм) ... (@место)? ... (@время)? ...
#### ... (@фильм) ... (@время)? ... (@место)? ...
#### ... (@время)? ... (@место)? ...(@фильм)  ...
#### ну тд, все комбинации
